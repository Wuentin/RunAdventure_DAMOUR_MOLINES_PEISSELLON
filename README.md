# RunAdventure
 //////////////////RUNADVENTURE//////////////////

Peripheral required: a keyboard;
Cavity keys:
S : Pull;
SPACE: Jump;
M: Soundless mode;
L : Sound off;
P: Pause;
ENTER : Select;
Directional arrows: Move;

Game Link: https://7x3h8.csb.app/
To be opened in a browser.

Good game!
